## Github Finder app

This is an app that allows you to search Github users.</br>
I built this project in order to practice using Hooks and Context API.

#### Install & Start

    $ git clone https://github.com/orvalho/github-finder
    $ cd github-finder
    $ npm install
    $ npm start
